## 项目背景

数据开发存在大量重复性工作（找表、写 SQL、评审代码、生成文档），借助 AI 工具可以大幅提升效率。目标是将 AI 工具深度嵌入数据开发工作流，而不是仅作表面工具使用。

## 角色与职责

- 团队 AI 工具体系负责人
- 设计并开发多个 Agent / MCP 工具
- 推动团队 AI Native 工作方式落地

## 技术方案

构建三层工具体系：
1. **MCP 服务层**：开发数据血缘查询 MCP、元数据查询 MCP，接入 Claude Desktop
2. **Agent 层**：AI 找表 Agent（自然语言 → 推荐数据表）、SQL 评审 Agent（自动检查规范）
3. **Skill 层**：封装常用数据开发 Prompt 为可复用 Skill，团队共享

## 成果

- 团队成员日均 AI 工具使用频次显著提升
- SQL 评审人工介入减少约 60%
- 新人上手数仓表体系时间缩短
